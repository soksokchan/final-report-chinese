    • 蘭潭校區

    蘭潭校區是嘉義大學的主要校區，設有農學院 、 理工學院 和 生命科學院 三學院。該校區坐落於蘭潭水庫旁，景致怡人、風景如畫，是大學生們憧憬的美好校園。其中特別的是, 蘭潭校區設有農場！農場裏養了各種各樣的牲畜，學生們利用此項優勢，自制嘉大牌子的乳制產品，為學校開創另一種特色。 蘭潭校區另一個引以爲傲之處，就是嘉大昆蟲館。昆蟲館擁有全球首創的「旋轉彩蝶柱」，1200多隻蝴蝶標本以雙股螺旋的方式圍繞在彩蝶柱中，不同的角度也會呈現出不同的色彩，真是美輪美奐。此外，該校區還有嘉義唯一一個設計構造不一樣的7-11, 以kano棒球為主題, 為校園添加了另一種風趣。另外，學校宿舍鄰近蘭潭風景區, 晚上還有音樂噴泉水舞燈光秀，是個浪漫約會的好去處。除此之外，風景區旁還有地標—月影潭心, 也是個熱門打卡的景點。

    民雄校區

    民雄校區設有師範學院 與人文藝術學院 。民雄校區占地雖然沒有蘭潭校區大，可是擁有著親切樸素的環境，學生們也都如環境一般親切十足，非常輕易就能讓人融入這個校園環境中。民雄校區範圍適中，校園建築都相距不遠，因此在校園中走動完全不費力，學生們更能因此賴床個幾分鐘，不用擔心會上課遲到呢！ 而民雄校區的特色景點就是鵝湖了。青葱又寧靜的環境，還有天鵝在湖裏遊蕩, 十分具有人文藝術氛圍，有如身處古代庭院一般,是個放鬆和尋找靈感的好地方。

    新民校區

    新民校區設有管理學院 與獸醫學院 。全台灣設有獸醫系的大學只有三間，嘉義大學就是其中一間。新民校區建有一家超大的獸醫院，開放給校外民衆帶自家的寵物來看病。學生們也可通過實際病例來學習。 新民校區是三個校區裡, 占地最小的校區，不過也是三大校區裡面最具有時尚年輕感的校區。新民校區裏有許多裝置藝術，常常都會看到喜歡攝影的朋友們來這裏取景拍攝。而新民校區位於嘉義市鬧區,學校附近到處都是吃店，完全就是美食天堂, 不怕沒吃的，就怕你的胃裝不下！

  • 高鐵 ：搭乘高鐵接駁車到高鐵桃園站 → 搭乘高鐵由高鐵桃園站到嘉義站 → 抵達後持高鐵票能夠免費搭乘BRT到嘉義火車站

  • 火車 ：桃園機場 → 搭乘桃園客運到桃園火車站 → 搭乘臺鐵由桃園站到嘉義火車站

  • 客運 ：桃園機場 → 搭乘統聯、國光客運到臺中朝馬站 → 由臺中朝馬站轉搭到嘉義轉運站

  • 高鐵 ：從小港機場搭捷運到新左營站 → 搭高鐵到嘉義高鐵站 → 高鐵接駁公車BRT到嘉義火車站

  • 火車 ：乘搭小港機場捷運到高雄火車站 → 乘搭臺鐵到嘉義火車站

  • 客運 ：小港機場搭捷運或計程車至高雄左營站 → 搭乘和欣客運到嘉義轉運站
