    機場到我嘉

    桃園

  • 高鐵 ：搭乘高鐵接駁車到高鐵桃園站 → 搭乘高鐵由高鐵桃園站到嘉義站 → 抵達後持高鐵票能夠免費搭乘BRT到嘉義火車站
  • 火車 ：桃園機場 → 搭乘桃園客運到桃園火車站 → 搭乘臺鐵由桃園站到嘉義火車站
  • 客運 ：桃園機場 → 搭乘統聯、國光客運到臺中朝馬站 → 由臺中朝馬站轉搭到嘉義轉運站

    • 高雄

  • 高鐵 ：從小港機場搭捷運到新左營站 → 搭高鐵到嘉義高鐵站 → 高鐵接駁公車BRT到嘉義火車站

  • 火車 ：乘搭小港機場捷運到高雄火車站 → 乘搭臺鐵到嘉義火車站

  • 客運 ：小港機場搭捷運或計程車至高雄左營站 → 搭乘和欣客運到嘉義轉運站
